Fixed Calculator 2.0

A Javascript calculator with fixed decimals and functions. I rewrote this calculator using math.js.
 Big numbers up to 10^(9*10^15), complex numbers, and high precision (50 digits) is supported.

Supported Operators and Functions
 +, -, *, /, ( ), ^, !, %, e, pi, tau, phi, i, deg, rad
sqrt(x), cbrt(x), log(x), log10(x), log2(x), log(x, y)
floor(x), round(x), ceil(x), abs(x), sign(x), exp(x)
re(x), im(x), conj(x), lcm(x, y, z...), gcd(x, y, z...)
sin(x), cos(x), tan(x), csc(x), sec(x), cot(x)
asin(x), acos(x), atan(x), acsc(x), asec(x), acot(x)
sinh(x), cosh(x), tanh(x), csch(x), sech(x), coth(x)
asinh(x), acosh(x), atanh(x), acsch(x), asech(x), acoth(x)
