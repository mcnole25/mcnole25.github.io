// INTRODUCTION //

Need a few ideas to create your custom block? Here's some cheat sheet arguments to help, and (hopefully) inspire you.
This list will probably be updated later.

// REPORTER BLOCK IDEAS //

If Else (AKA: Ternary Operator)
return args.D ? args.A : args.B

Letters A to B of String
return String(args.C).substring(args.A - 1, args.B)

Clamp Value
return args.A <= args.B ? args.B : args.A >= args.C ? args.C : args.A

Replace Substrings
return args.C.toString().replace(new RegExp(args.A, "gi"), args.B)

Repeat String
return String(args.A).repeat(args.B)

Number to Unicode
return String.fromCharCode(args.A)

Reverse Text
return String(args.A).split("").reverse().join("")

Triple Join
return String(args.A).concat(String(args.B).concat(String(args.C)))

// BOOLEAN BLOCK IDEAS //

Strictly Contains
return String(args.A).includes(String(args.B))

Percent Chance
return Math.random() < args.A / 100
